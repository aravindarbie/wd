# Wedding Invitation

Open `index.html` in a browser to preview the invitation.

Edit `config.js` to change the couple names, wedding date, time, venue, map link, background music, gallery photos, invitation message, and RSVP destination.

For RSVP delivery:
- Set `whatsappNumber` to a WhatsApp number with country code and no plus sign.
- Or set `rsvpEmail` to use an email draft.
- If both are blank, the form opens WhatsApp share with the RSVP text.
